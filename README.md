# 谨遵指令

边狱巴士指令模拟器 —— 模拟"食指"组织下达指令的桌面应用。

在游玩《边狱巴士》镜牢模式时，软件生成指令并通过传呼机悬浮窗展示。遵守或违抗指令被记录为"指令加护"与"业"。

## 下载

从 [Releases](../../releases) 下载最新版 `谨遵指令 x.x.x.exe`，双击运行即可。

## 功能

- **人格 & EGO 管理**：179 条人格、109 条 EGO，支持池、筛选、批量操作
- **镜牢指令生成**：13 个游戏环节，覆盖编队、星光、饰品、卡包、路线、战斗、事件、商店、关底
- **传呼机悬浮窗**：透明置顶，打字机动画，指引导航
- **加护 & 业**：遵守指令 +1 加护，违抗 +5 业，影响后续指令的合理性判断
- **成就系统**：12 个成就，条件树自动检测
- **编队窗口**：点击选号，拖拽排序
- **自由指令**：地牢中随机弹出趣味指令
- **业恐吓**：业超标时全屏警告效果
